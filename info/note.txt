-- Lệnh tạo worktree cho commit cũ của git để làm báo cáo:
+ xem all commit: 
-> git log --oneline --graph --decorate --all
+ Tạo folder lưu commit mình muốn: 
-> git worktree add ../repo-commit <7 kí tự log SHA_or_REF>
+ xem tạm thời 
-> git switch --detach <7 kí tự log SHA_or_REF>
